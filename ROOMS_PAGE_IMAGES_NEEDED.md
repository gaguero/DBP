## Image Specifications

### Format
- **Preferred**: `.avif` (best compression, modern browsers)
- **Alternative**: `.jpg` or `.webp` (for compatibility)

### Dimensions
- **Hero Image**: 1920x800px (16:6.67 ratio) or similar wide format
- **Section Images**: 1200x800px (3:2 ratio) or 1600x900px (16:9 ratio)
- **Package Images**: 800x600px (4:3 ratio) for cards

### SEO Naming Convention
- Use lowercase with hyphens
- Include key descriptive terms
- Include location context (dolphin-bay, bocas-del-toro)
- Example: `swim-platform-crystal-clear-water-snorkeling-dolphin-bay.avif`
