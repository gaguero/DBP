## Technical Specifications

### Dimensions
- **Width**: 1920px minimum
- **Height**: 800px (for 50vh section with min-h-[400px] max-h-[600px])
- **Aspect Ratio**: 2.4:1 or similar wide format

### Format
- **Preferred**: `.avif` (best compression)
- **Alternative**: `.jpg` (high quality, 90%+ compression)

### Composition Tips
- Leave space for potential text overlay (though current design doesn't use it)
- Ensure key elements (cabanas, bay) are visible on mobile (center-focused)
- Consider rule of thirds for visual interest
- Ensure good contrast for readability if text is added later
